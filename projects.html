Project Portfolio

Here are the projects I've done so far. Feel free to explore.

Kitchenware Classification

In this project, we used Deep Learning to predict if an image is a spoon, knife, plate, cup, glass, or a fork. A total of 4445 images was used to train the model. The model's accuracy is around 95.5%

Radiant or Dire?

Who do you think will win? Light or Dark? Good or Bad? RADIANT OR DIRE? In this project, we used Machine Learning to predict the winner of a Dota 2 match using pre-game features: region, cluster, time, average winrates, players skills, and player heroes.

Spotify API Data Pipeline

In this project, data is captured from the Spotify Web API using the python request module. The data collected from the Spotify Web API is stored in CSV file, JSON file, and SQL Serve database. ETL jobs are written in python and scheduled in airflow to run every 12 midnight (UTC).
